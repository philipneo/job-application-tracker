# Job Application Tracker

A lightweight browser app for tracking job applications, follow-ups, contacts, and interview status.

## Features

- Add job applications with company, role, status, and date
- Filter by application status
- Store entries in localStorage
- Track follow-up notes
- Simple dashboard counts

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Goals

This project demonstrates basic CRUD behavior, browser storage, DOM updates, and practical workflow design.
